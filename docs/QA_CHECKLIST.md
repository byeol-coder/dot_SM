## 9. 자동화 QA 스크립트 (Playwright 기반)

```bash
# 설치
npm install -D @playwright/test axe-playwright

# 실행 (dist 기준)
npx playwright test tests/a11y.spec.ts
```

주요 테스트:
- `axe.run('#root')` → WCAG 2.2 위반 0건
- F1~F4 키 이벤트 시뮬레이션 → aria-live 텍스트 변화 확인
- `prefers-reduced-motion: reduce` → `--dur` 변수 0ms 적용 확인
- 모달 포커스 트랩 순환 확인
